Esses exercícios têm como objetivo treinar conceitos fundamentais da linguagem C, como leitura de dados, vetores, matrizes, structs, ponteiros, funções e alocação dinâmica. 

O primeiro exercício, o programa lê três palavras e imprime na ordem inversa. Você armazena cada palavra em uma variável e apenas muda a ordem na hora de exibir.

O segundo exercício, é utilizado um vetor de 5 posições para armazenar números inteiros digitados pelo usuário. Após preencher o vetor, o programa imprime os valores de trás para frente. Isso reforça o conceito de índices de vetor, que começam em zero, e o uso de laços para percorrer dados em ordem inversa.

O terceiro exercício trabalha com matrizes 3x3. O programa lê os valores usando dois laços for, multiplica cada elemento por 5 e imprime o resultado.

O quarto exercício, o objetivo é demonstrar a propriedade da matriz identidade. Primeiro, é criada uma matriz identidade. Depois, essa matriz é multiplicada pela matriz original. O resultado da multiplicação deve ser exatamente igual à matriz original.

O quinto exercício, o programa realiza a multiplicação de um vetor por uma matriz. Para cada coluna da matriz, é feito um cálculo que multiplica os elementos do vetor pelos elementos correspondentes da coluna, somando os resultados. 

O sexto exercício introduz o uso de struct para armazenar dados mais complexos, neste caso, informações de alunos. O programa lê 10 alunos e os separa em dois grupos: aprovados e reprovados, com base na média mínima de 5.0. 

No sétimo exercício, o programa também utiliza struct, mas agora para armazenar dados de livros. Após ler os dados, o usuário informa um título para busca, e o programa compara esse título com os cadastrados usando a função strcmp. 

O oitavo exercício foca em ponteiros. Um vetor é criado, mas o acesso aos seus elementos é feito apenas com aritmética de ponteiros, sem usar a notação tradicional de índice. O programa lê os valores e imprime o dobro de cada um. 

No nono exercício, o programa utiliza uma função que recebe três valores por referência. A função ordena esses valores diretamente nas variáveis originais e retorna 1 se todos forem iguais ou 0 caso contrário. 
No décimo exercício trabalha com alocação dinâmica de memória. O programa lê um número n e usa malloc para criar um vetor de alunos dinamicamente. Em seguida, uma função percorre esse vetor usando ponteiros e retorna o aluno com a maior nota.
